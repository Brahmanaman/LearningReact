# React Styling Type using css

1. Inline Style --> use camelCase in style `style = {{color : 'blue', fontSize='16px'}}`
2. External Style --> seperate file for style
3. Css Module --> similar to External Style but it is component base `MyComponent.module.css`
4. Styled Components --> it allow us to write a css directly within javascript component. This approach is known as CSS-in-Js.
5. External Css Library/FrameWork --> Material UI, React bootstrap, tailwind css.
